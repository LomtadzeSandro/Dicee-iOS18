Dicee is a simple iOS application built with **Swift** and **UIKit** that simulates rolling two dice.  
The user taps the "Roll" button, and the app randomly displays images of two dice, demonstrating basic **UI interaction** and **randomization logic**.

This project demonstrates **IBOutlet/IBAction usage, UIImageView updates, arrays, and random element selection** in iOS.

---

## Features

- Tap the "Roll" button to roll two dice
- Displays two random dice images
- Clean and responsive UI
- Demonstrates basic UIKit interaction and logic
- Lightweight and beginner-friendly

---

## Tech Stack

- Language: Swift  
- Framework: UIKit  
- Architecture: MVC  
- IDE: Xcode  
- iOS Target: iOS 13+
